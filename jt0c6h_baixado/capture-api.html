Como capturar respostas da API do site

Passo 1: Abra o site no Chrome

Abra https://jt0c6h.com/main/inicio/ em uma nova aba.

Passo 2: Abra o DevTools

Pressione F12 ou Cmd+Option+I (Mac) para abrir as Ferramentas do Desenvolvedor.

Passo 3: Aba Network

Clique na aba Network (Rede). No filtro, digite trpc.

Passo 4: Recarregue a página

Recarregue a página (Cmd+R ou F5) e aguarde carregar.

Passo 5: Copie as respostas

Para cada requisição tRPC (home.list, home.popularGames, carouselConfig.list, etc.):

  1. Clique na requisição
  2. Abra a aba Response (Resposta)
  3. Clique com o botão direito → Copy → Copy value

Ou: clique com o botão direito na lista de requisições → Save all as HAR with content e envie o arquivo.

Endpoints importantes para capturar

  • home.list - lista de jogos (erro: .map)
  • home.popularGames - jogos populares (erro: .filter)
  • carouselConfig.list - carrossel (erro: .content.reduce)
  • home.marquee ou similar - marquee (erro: .content.reduce)

Alternativa: Usar servidor proxy

Execute o servidor proxy que tenta buscar dados reais do site:

cd /Volumes/midascod/chinesa
python3 server-proxy.py

Depois acesse http://localhost:3001/main/inicio/

Se o proxy conseguir conectar ao site, os dados reais serão usados e os erros devem desaparecer.
