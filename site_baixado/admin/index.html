Manipulação de Indicações (Sistema 3/1)

Atenção: Este sistema controla quantas indicações são reais e quantas a casa «rouba». Exemplo: 3/1 significa que a cada 3 indicações dadas ao afiliado, 1 é subtraída pela casa.
Ativar Manipulação de Indicações Quando ativado, o sistema irá manipular as indicações conforme configurado abaixo. Quando desativado, todas as indicações serão contabilizadas normalmente.

Dar Indicações
